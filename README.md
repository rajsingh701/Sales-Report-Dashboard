# 📊 Interactive Dynamic Sales Dashboard in Excel

## 📌 Project Overview

This project showcases an **Interactive Dynamic Sales Dashboard** built entirely in **Microsoft Excel**. The dashboard transforms raw sales data into meaningful business insights through interactive visualizations, KPI tracking, and dynamic filtering.

Designed with business intelligence principles, this dashboard enables users to analyze sales performance across different regions, product categories, and years, supporting data-driven decision-making.
